
# 📚 Bookstore UI – Modern Interactive Web Experience

## 🔹 Overview
A visually rich and interactive bookstore website built with modern UI/UX principles, smooth animations, and responsive design.

## 🔹 Description
This project showcases a premium digital bookstore experience with elegant layouts and immersive interactions.  
It focuses on storytelling through UI using smooth transitions, hover effects, and section-based navigation.  
Designed to deliver both aesthetic appeal and functional usability across all devices.

---

## ✨ Features
- 📱 Fully Responsive Design  
- 🎬 Smooth Animations & Transitions  
- 🧭 Section Navigation with Scroll Effects  
- 🖱️ Interactive Hover Effects  

---

## 🧩 Sections

- **Hero Section:** Engaging landing with tagline, CTA buttons, and scroll indicator.  
- **About Us:** Displays vision, mission, and brand story in interactive book layout.  
- **Bookshelf:** Showcases curated books with category filtering and hover effects.  
- **Reviews:** Highlights customer testimonials with slider interaction.  
- **Visit Store:** Provides location, timings, and contact details.  
- **Footer:** Includes quick links, policies, and newsletter subscription.

---

## 🛠️ Tech Stack

- ⚡ Next.js  
- 🎨 Tailwind CSS  
- 🎞️ Framer Motion  
- 🧩 shadcn/ui

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/bookstore-ui.git

# Navigate into the project
cd bookstore-ui

# Install dependencies
npm install

# Run development server
npm run dev
````

---

## 🚀 Usage

Open `http://localhost:3000` in your browser to view the project.

---

## 📌 Notes

* Ensure Node.js is installed (v18+ recommended)
* Tailwind config should be properly set
* Add your environment variables if needed
